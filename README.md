# GIT使用

* 克隆

  ```shell
  # 克隆
  git clone https://github.com/myliwenbo/SpringDemo.git
  ```

* 添加要提交的信息

  ```shell
  # . 代表是当前文件下的所有内容都提交，通配符
  git add .  
  ```

* 提交到本地仓库

  ```shell
    git commit  -m  ”提交信息”
  ```

* 提交到`github`上，需要添加用户名和密码

  ```shell
  git push -u origin master
  ```

# 删除

* 显示所有本地分支

  ```shell
  git branch
  ```

* 删除即可，用命令删除即可。

  ```shell
   rm -rf
  ```
